# DSA Learning

This learning repository focuses on core algorithmic patterns used in problem solving:

- `hashing`
- `two-pointer`
- `sliding-window`
- `binary-search`

Each folder contains a README explaining the technique, how to recognize the pattern, and sample problem examples.

## How to use this repository

1. Start with the `README.md` in each subfolder.
2. Read the pattern identification tips.
3. Walk through the example problems and solutions.
4. Practice by applying the technique to new problems.
